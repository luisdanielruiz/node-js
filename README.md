# node-js
A recipe CRUD API app built on node js , express js and mongodb

## Requirements

* NodeJS
* Mongo DB
* express JS framework


## Installation

#### 1. Clone the Github Repositorty'

use npm or yarn
  using npm

         
      $ npm install
       
   
   using yarn
   
     
      $ yarn install
  

#### 2. Run the Application

  using npm

       
        $ npm run
       
   
  using yarn
   
        
        $ yarn run
    
#### 3. Running the Tests

  using npm

       
         $ npm run test --coverage
       
   
  using yarn
   
        
        $ yarn run test --coverage
